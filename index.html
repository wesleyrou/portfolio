Projects

Binaural

Binaural generates binaural beats and audio visualizations to help users achieve altered states of consciousness.

There are other apps that allow a user to relax to soothing sounds or use sound to get energized for the day, but none of them have a visual component. With this project, my team wanted to create an app that would use the power of binaural beats to stimulate brain waves while also adding a visual component that would simulate the waveforms of the beats.

Tech: HTML5, CSS, React, Javascript, SineWaves.js, Web Audio API, Express, knex..js, PostgreSQL, Mocha
